# WHAT IS THIS?
> A vue project.

> A todo list demo.

Just download the project,and run
```shell
npm install cooking-cli -g;
```
enter the project directory  and run
 ```shell
 npm install ;
 ```

# Development
open dev-server:
```shell
cooking watch
```
checkout at localhost:8989

open mock-server:
```shell
node mock/server
```
The mock server is on port:3000



# Production
```
cooking build
```
